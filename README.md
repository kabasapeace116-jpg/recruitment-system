# MAHIKENG INVESTMENTS - Recruitment Management System

A comprehensive recruitment management system for placing candidates from Uganda to the Arab world.

## Features

- **Admin Dashboard**: Manage candidates, clients, and track selections
- **Client Portal**: Clients can browse, filter, and select candidates
- **Candidate Management**: Register candidates with personal info, documents, and video introductions
- **Selection Tracking**: Track which clients selected which candidates
- **Advanced Filters**: Filter by age, religion, job category, experience
- **Responsive Design**: Works on desktop, tablet, and mobile

## Tech Stack

- Python 3.14+
- Django 6.0
- SQLite (Development) / MySQL (Production)
- HTML5, CSS3, JavaScript
- Chart.js for analytics
